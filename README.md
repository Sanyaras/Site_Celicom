# Целиком — WMS-система для фулфилментов

Сайт-визитка для WMS-системы "Целиком".

## Технологии

- HTML5
- CSS3 (CSS Custom Properties, Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- Google Fonts (Inter)

## Структура проекта

```
Site_Celicom/
├── index.html          # Основной HTML файл
├── styles.css          # Стили
├── script.js           # JavaScript
├── assets/
│   └── images/         # Изображения (добавить позже)
└── README.md
```

## Запуск

Открыть `index.html` в браузере или использовать локальный сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js (npx)
npx serve
```
